# Study Goal Tracker - Frontend

A modern web application for tracking study goals, managing tasks, notes, and student attendance.

## Features

- 📊 **Dashboard** - Overview of tasks and progress
- ✅ **Task Management** - Create, assign, and track academic tasks
- 📝 **Notes Section** - Write and manage study notes
- 👥 **Contact Management** - Manage student contacts
- 🧮 **Calculator** - Built-in calculator tool
- 🤖 **AI Assistant** - Interactive assistant for help
- 🌓 **Dark Mode** - Toggle between light and dark themes

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 5.3.0
- Font Awesome 6.4.0

## Getting Started

1. Clone this repository or download the files
2. Open `index.html` in your web browser
3. No build process required - it's a static website

## Usage

Simply open any HTML file in a modern web browser. The application uses local storage for data persistence.
